Django Full Stack Developer Needed

Fixed Price: $1,500 - $3,000 Expert Level Posted 2 hours ago

United States | Project Length: 1-3 months

Job Description

We are looking for an experienced Django full-stack developer to build a comprehensive freelance marketplace platform similar to popular freelancing websites. The ideal candidate should have strong experience in both backend and frontend development.

Requirements
  • 5+ years of experience with Django and Python
  • Strong knowledge of Django REST Framework
  • Experience with PostgreSQL database design and optimization
  • Proficiency in frontend technologies (HTML, CSS, JavaScript, Bootstrap)
  • Understanding of authentication and authorization mechanisms
  • Experience with payment gateway integration (Stripe, PayPal)
  • Knowledge of real-time features using WebSockets or similar technologies
  • Familiarity with Git version control
Project Scope

The platform should include the following features:

  • User registration and authentication (Client & Freelancer roles)
  • Job posting and browsing functionality
  • Proposal submission system
  • Messaging system between clients and freelancers
  • Payment integration and escrow system
  • Rating and review system
  • Admin dashboard for platform management
  • Responsive design for all devices
